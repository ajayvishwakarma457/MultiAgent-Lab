# Git Conventions

## Commit Format
type(scope): subject

## Types
- feat: New feature
- fix: Bug fix
- refactor: Code refactor
- style: Formatting
- test: Tests
- docs: Documentation

## Examples
- feat(todo): add priority filter
- fix(api): handle timeout
- refactor(components): extract TodoItem
