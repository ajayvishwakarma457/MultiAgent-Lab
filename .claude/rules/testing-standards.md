# Testing Standards

## Unit Tests
- Test individual functions
- Test edge cases
- Mock external dependencies

## Component Tests
- Props validation
- User interactions
- Conditional rendering

## Integration Tests
- Multiple components together
- State management
- Async operations

## Coverage Targets
- Statements: 80%+
- Branches: 75%+
- Functions: 80%+
- Lines: 80%+
